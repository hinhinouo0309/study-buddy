# Study Materials

Place your lecture note PDFs in this folder so the `Research Agent` can discover them through MCP.

## Example

```text
study_materials/week_10_note.pdf
```

## Demo prompt

After adding a PDF, run `main.py` and try:

```text
Use week_10_note.pdf to generate revision points, key concepts, and study questions, then add them into my study plan.
```

## Notes

- PDF files are **not included** in this repository by default. Add your own lecture notes locally.
- Do not upload copyrighted course materials to a public GitHub repo unless you have permission.
- Only files inside `study_materials/` can be read by the MCP filesystem server.
